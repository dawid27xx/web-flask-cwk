# Description

LeedsHub is a social media site catered for University of Leeds students. The website offers a place for Leeds Students to unite and interact through a public feed page or specific groups.

# Usage 

Dependencies: 
- Flask
- Werkzeug
- SQLAlchemy
- wtforms
